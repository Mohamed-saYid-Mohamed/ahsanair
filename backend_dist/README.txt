# How to Install the Backend (XAMPP)

1.  **Locate the `backend_dist` folder** in this project.
2.  **Copy** the `api` folder inside it.
3.  **Go to your XAMPP installation** (usually `C:\xampp\htdocs`).
4.  **Create a new folder** named `ahsan-api`.
5.  **Paste** the `api` folder inside `ahsan-api`.

Your structure should look like this:
`C:\xampp\htdocs\ahsan-api\api\track.php`
`C:\xampp\htdocs\ahsan-api\api\contact.php`
...etc.

6.  **Start Apache** in XAMPP Control Panel.
7.  The website should now be able to talk to the backend!
